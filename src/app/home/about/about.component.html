I’m a front-end developer, working on web and desktop apps in post-brexit UK. I specialise in HTML, CSS and JavaScript for the web, and a variety of different programming languages and interface frameworks for desktop and mobile.

When I’m not lost in a screenful of code, you’ll probably find me drawing, taking photos, reading up on things or tinkering with creating objects I’ve created using my 3D printer.

Creating usable software which is a delight to use is no easy task, and I am constantly learning new things to keep up with the industry. I aim to produce software which puts the user first and in direct control, whilst still being powerful and feature rich.
